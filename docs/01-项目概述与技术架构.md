# 家庭影视平台 - 项目概述与技术架构

## 一、项目简介

### 1.1 项目概述
**项目名称**: HomeVideoWeb（家庭影视平台）
**项目描述**: 类似网易爆米花风格的家庭私有影视媒体库管理平台
**核心功能**: 电影/剧集管理、自动刮削元数据、多源扫描、在线播放、智能搜索

### 1.2 项目特点
- **多源支持**: 支持本地路径、SMB共享、NFS共享三种媒体源类型
- **自动刮削**: 集成OMDb、豆瓣、TMDB三大元数据源
- **智能搜索**: 支持中文名称、拼音首字母、英文名称搜索
- **在线播放**: 支持主流视频格式直接播放，MP4/MKV/AVI等
- **进度同步**: 自动记录播放进度，多端同步

## 四、构建与部署

### 4.1 构建命令

#### 根目录 package.json
```json
{
  "scripts": {
    "dev": "concurrently \"npm run dev:server\" \"npm run dev:client\"",
    "dev:server": "nodemon --watch server --ext ts --exec ts-node server/index.ts",
    "dev:client": "cd client && npm run dev",
    "build": "npm run build:server && npm run build:client",
    "build:server": "tsc --project server/tsconfig.json",
    "build:client": "cd client && npm run build",
    "start": "node dist/server/index.js"
  }
}
```

#### 客户端 package.json
```json
{
  "scripts": {
    "dev": "vite",
    "build": "tsc && vite build",
    "preview": "vite preview"
  }
}
```

### 4.2 环境变量配置

创建 `.env` 文件（参考 `.env.sample`）:

```env
# 服务端口
PORT=3000

# OMDb API Key（获取IMDb评分）
OMDB_API_KEY=your_omdb_api_key

# 豆瓣 API Key（获取中文元数据）
DOUBAN_API_KEY=your_douban_api_key

# TMDB API Key（增强海报和元数据来源）
TMDB_API_KEY=your_tmdb_api_key

# 数据库路径
DB_PATH=./data/movies.db

# 海报缓存目录
POSTER_DIR=./data/images
```

### 4.3 启动方式

#### Windows 用户（推荐）
直接双击运行 `start.bat`，脚本会自动完成:
1. 端口检测并释放3000
2. 安装依赖
3. 生产构建
4. 启动服务

#### 手动启动
```bash
# 开发模式
npm run dev

# 生产模式
npm run build
npm start
```

### 4.4 访问地址

- **本地访问**: http://localhost:3000
- **局域网访问**: http://192.168.x.x:3000（自动检测）
